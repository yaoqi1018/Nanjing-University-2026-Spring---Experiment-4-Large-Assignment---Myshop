# 我的网购小店

一个基于 C++ 的命令行电子商务系统，支持用户注册登录、商品浏览搜索、购物车、订单管理、收藏夹、商品评价、个人中心等功能，同时提供管理员后台进行商品、订单、用户和评价的管理。

## 功能概览

### 用户端

| 模块 | 功能 |
|------|------|
| **用户系统** | 注册、登录、修改密码、查看个人信息与会员等级 |
| **商品浏览** | 查看全部商品、按名称搜索、按分类筛选、按价格区间筛选、按分类价格排序、查看商品详情与评价 |
| **购物车** | 加入购物车、查看购物车、修改数量、删除商品、购物车下单 |
| **订单管理** | 查看我的订单、确认收货、申请退款 |
| **收藏夹** | 收藏商品、查看收藏、取消收藏、从收藏加入购物车 |
| **个人中心** | 查看个人信息和会员等级、修改密码、修改收货地址、充值余额、浏览历史 |
| **商品评价** | 对已购买的商品撰写评价 |

### 管理员端

| 模块 | 功能 |
|------|------|
| **商品管理** | 查看、添加、修改、删除商品 |
| **订单管理** | 查看全部订单、修改订单状态、维护物流信息 |
| **用户管理** | 查看所有用户、重置用户密码 |
| **评价管理** | 查看全部评价、回复评价、删除评价 |
| **退款审核** | 处理用户提交的退款申请 |

## 数据结构

### 用户 (User)
```
用户名 | 密码 | 手机号 | 累计消费金额 | 支付余额 | 收货地址
```

- **会员等级**：普通会员（累计消费 < 1000）、银卡会员（1000-5000）、金卡会员（5000-10000）、钻石会员（≥ 10000），不同等级享受不同折扣

### 商品 (Product)
```
商品编号 | 名称 | 分类 | 单价 | 库存
```

### 订单 (Order)
```
订单号 | 用户名 | 商品编号 | 商品名称 | 数量 | 实付金额 | 状态 | 物流信息
```

订单状态包括：已支付、已发货、已收货、退款中、已退款。

## 环境要求

- **编译器**：g++ (MinGW-w64)，推荐使用 `D:\mingw64\bin\g++.exe`
- **操作系统**：Windows（使用 `SetConsoleOutputCP(CP_UTF8)` 支持中文输出），也可在 Linux/macOS 下编译运行
- **IDE**：Visual Studio Code（已配置 `.vscode/tasks.json`）

## 编译与运行

### 方式一：命令行编译

```bash
# 编译所有源文件
g++ -o main.exe main.cpp User.cpp Product.cpp Cart.cpp Order.cpp Favorite.cpp History.cpp Review.cpp -fexec-charset=UTF-8

# 运行
./main.exe
```

### 方式二：VSCode 一键编译运行

在 VSCode 中打开 `main.cpp`，按 `Ctrl+Shift+B` 使用预配置的 g++ 构建任务，然后运行生成的 `main.exe`。

> **注意**：请根据你本地 g++ 的安装路径修改 `.vscode/tasks.json` 中的 `command` 字段。

## 使用说明

1. 启动程序后，进入主菜单选择「用户注册」创建账号
2. 注册成功后，使用「用户登录」进入用户功能区
3. 在商品功能中浏览商品，将心仪商品加入购物车
4. 在购物车中确认商品和数量后下单购买
5. 可在订单功能中查看订单状态、确认收货或申请退款
6. 管理员通过「管理员登录」进入后台管理系统

### 会员折扣规则

| 会员等级 | 累计消费金额 | 折扣 |
|----------|-------------|------|
| 普通会员 | < 1000 元 | 无折扣 |
| 银卡会员 | 1000 ~ 5000 元 | 95 折 |
| 金卡会员 | 5000 ~ 10000 元 | 90 折 |
| 钻石会员 | ≥ 10000 元 | 85 折 |

## 数据持久化

所有数据以纯文本格式（`.txt`）存储在项目根目录，程序启动时自动加载，操作完成后自动保存。每个字段之间使用特定分隔符，方便阅读和维护。
